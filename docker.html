Basic Docker command!


Instalation

Follow the official website documenation for installation of docker, as per your evniroment

Resources

confirm your docker installation

docker run hello-world



Downloading only

use the following commands to download the images EXAMPLE

docker pull debian

docker pull redhat/ubi8

docker pull ubuntu



Initalinze a container image

use the following command to run an interactive environment EXAMPLE

docker run -it debian


Terminate a container image

user the following command to terminate the interactive environment

docker stop CONTAINER ID


use the following command to Check running processes

docker ps


use the following command to lists all the containers

docker container ls -a



List docker Images

use the following command to Check for locally available docker images

docker images



Execute with interactive shell environment

use the following command on a running containerEXAMPLE

docker container exec -it f13bc3618719 bash



Check container properties

use the following command to inspect assigned container

INFO == debian or CONTAINER ID

docker inspect INFO


Logs access

use the following command to list the containerization logs

docker ps -a


use the following command to list the selected image log

docker logs CONTAINER ID



Clean up

use the following command to clean you images

All

docker container prune -f
