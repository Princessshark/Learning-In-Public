Basic Git Commands!


use the following command to see instruction manual

git help

specific help for commands EXAMPLE init

git help init


use the following command to check your existing configuration

git config --list


use such following commands to specific existing configuration

git config user.name

git config user.email



Start by configuring your git account

use the following commands to configure your git account

git config --global user.name "Jhon Doe"


git config --global user.email "doejhon@yahoo.com"



Create a New git Project

use the followin command to initailze an empty git reposotary

git init



Project status

use the following command to check the status of file and directories

git status

Untracked FIles

Added in Staging Area



Prior to a commit, we need to add the files or directory in a staging area

use the following command to add all the files or directory to the staging area

git add .


use the following command to remove files or directories from the staging area

git restore --staged "fileName.fileType"


use the following command to add files or directory from a designate PATH

git add PATH/fileName.fileExtention



To Commit with a bookmark

use the following command, after adding the changes into the staging area

git commit -m "my message"



Access the history of all your commints

use the following command to access the entire log of your commits

git log



Restore or updates

use the following commands


Restore log back in time

git reset 'commitHash'


Restore deleted files or directory

git checkout HEAD ./PATH



Store files or directories on the background withought updating the log

use the following command with git add command, as the selector

git stash


Retrive files or directories from the background location

use the following command retrive stash data from the background

git stash pop


Permanently clear files or directories from the background location

use the following command remove from stash

git stash clear
